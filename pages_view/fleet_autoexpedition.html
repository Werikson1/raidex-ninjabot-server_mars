Multi expedition

Fleets : 0/57
Expeditions : 0/14
  • With the "multi expedition" feature, you can create multiple expedition missions with one click.
  • Using the menu below, you can select the number of expeditions, duration of expeditions and fleets you want to send to expedition.
  • The number of ships selected is divided by the number of expeditions. For example, if 100 ships and 10 exploration numbers were selected, 10 ships will be sent to each exploration.
  • (Attention: If there is not enough deuterium available for all selected ships, a part of the fleet will be sent.)
